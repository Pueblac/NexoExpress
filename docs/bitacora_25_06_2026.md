## 🏛️ Decisiones de Arquitectura / Ecosistema

- **`start.py`** es el nuevo punto de entrada oficial de BitácoraExpress en Linux. Resuelve el problema de encoding del carácter `á` en la ruta al usar subprocess directamente con Python.
- **`firestoreSet`** añadido a `firebaseAdmin.ts` — permite escribir un documento con ID específico (PATCH con updateMask). Es el patrón correcto para `sintesis/{actaId}`.
- **Síntesis no es bloqueante**: si falla el guardado de `sintesis/`, el acta se guarda igualmente. La síntesis se pobla con lo que Gemini devuelva (transcripción, temas, preguntas).
- **`PROJECT_BASE_DIR`** en `.env` (en `.gitignore`) — cada desarrollador configura su ruta local sin afectar el repo.

## 🐛 Issues Encontrados / Workarounds

- **Problema de encoding `á`**: subprocess no puede ejecutar binarios en rutas con caracteres Unicode compuestos (NFC vs NFD). **Workaround**: `start.py` usa `import uvicorn; uvicorn.run(...)` directamente dentro del mismo proceso Python del venv, evitando subprocess con paths problemáticos.
