## 🏛️ Decisiones de Arquitectura / Ecosistema

- **NexoExpress** es el nombre oficial del coordinador. Reemplaza "Coordinador_ActaExpress" como identidad conceptual (la carpeta mantiene su nombre por compatibilidad con el workspace).
- **Regla de prototipado:** Web siempre primero. Android no implementa features que Web no tiene.
- **Esquema Firestore unificado:** `schemas/firestore_schema.md` es el contrato. Ningún proyecto crea colecciones sin documentarlas ahí.
- **BitácoraExpress usa prefijo `be_`** en sus colecciones Firestore para distinguirlas de las de ActaExpress.
- **`trackeandomipega`** (repo GitHub) confirmado como proyecto de videojuego personal sin relación con el ecosistema. Excluido del radar.
